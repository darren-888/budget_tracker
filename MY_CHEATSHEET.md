## 🚀 How to Push Updates (run in Terminal)

Every time I make changes to the app, run these 3 commands:

```
cd /Applications/XAMPP/xamppfiles/htdocs/BUDGET_TRACKER
git add .
git commit -m "describe what you changed"
git push
```

Then wait ~1 minute → Vercel auto-updates the live site. Done!

---

## 💻 How to Run the App Locally (for testing)

```
cd /Applications/XAMPP/xamppfiles/htdocs/BUDGET_TRACKER
npm run dev
```

Then open: http://localhost:5173

## 📱 How to Install on My Phone

### iPhone (Safari only!):
1. Open the live link in Safari
2. Tap Share button (box with arrow ↑)
3. Tap "Add to Home Screen"
4. Tap "Add"

### Android (Chrome):
1. Open the live link in Chrome
2. Tap ⋮ menu → "Add to Home Screen"
3. Tap "Add"
